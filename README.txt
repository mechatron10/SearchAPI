This is the API that can perform the search operation on the fetched comments from youtube videos 
To use this API follow these steps-
1 Create a virtual environment  using 
    python -m venv name_of_environment 
     or 
    python3 -m venv name_of_environment
2 Activate the virtual environment.
    name_of_environment\Scripts\activate
3 Navigate to the project directory containing the requirements.txt file.

4 Install the project dependencies by running:
     pip install -r requirements.txt
5 Now run the API using the command 
  flask run
